Module temple::sprite[−][src]

Sprite and sprite type config management

Usage

Sprite types and sprites are loaded on game boot. The textures supplied by the manifest's SpriteEntry list are loaded into Bevy's [AssetServer] for later use, and then are converted to a GameSprite with a reference to the loaded texture and associated attributes.
